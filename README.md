# Abstitch — E-commerce Platform

Built with Next.js 14, TypeScript, Supabase, and Stripe.

---

## Getting Started

### 1. Clone & Install
```bash
git clone <repo-url>
cd abstitch
npm install
```

### 2. Environment Variables
Get the `.env.local`

### 3. Run Locally
```bash
npm run dev
```

- Site: [http://localhost:3000](http://localhost:3000)
- Admin: [http://localhost:3000/admin](http://localhost:3000/admin)

---

## Tech Stack
Next.js 14 · TypeScript · Tailwind CSS · Supabase · Stripe · Resend · Vercel
